## Practical Catalog Rules for Upload Quality
These rules are used by DK Trendify to improve listing acceptance and conversion quality.

### Product Image Rules
- Use clean product-first photos.
- Avoid heavy text overlays, price stickers, or promotional text on image.
- Avoid visible watermarks or third-party logos when not permitted.
- Keep product framing clear and centered.
- Avoid extra props or humans unless category requires lifestyle context.
- Avoid distorted, stretched, or rotated-looking images.
- Avoid blurry images.

### Title Rules
- Keep title clear and searchable.
- Include product type + key attribute + use context.
- Avoid keyword stuffing and irrelevant terms.

### Description Rules
- Keep short and buyer-friendly.
- Use clear bullet points for material, utility, and size/fit details.
- Highlight practical value and daily use case.
- Avoid misleading claims.

### Category and Attribute Rules
- Select the closest category to avoid discovery mismatch.
- Fill core attributes relevant to category filters.
- Ensure title, attributes, and image all represent the same product.

### Compliance and Risk Rules
- Avoid restricted or prohibited items.
- Avoid infringing branding and unverified brand claims.
- Prioritize categories with historically lower return friction.
- Use practical, expectation-aligned descriptions to reduce returns.
